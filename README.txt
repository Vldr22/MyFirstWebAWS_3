                                    УЧЕБНЫЙ ПРОЕКТ №1 "САЙТ СО СПИСКОМ ФАЙЛОВ"

Проект реализует базовый функционал загрузки/скачивания файлов на сайте с использованием хранилища YandexStorage S3.
Помимо ознакомления с AWS, целью проекта было ознакомление на базовом уровне с технологиями:

1. Создание, настройка и запуск Object Storage в рамках java spring приложения
2. Работа с Html и CSS в java. Использование библиотеке thymeleaf
3. Реализация пагинации
4. Security Spring c простой авторизацией и хеширования паролей
5. Input/Output механизм в контексте обмена файлами с AWS
